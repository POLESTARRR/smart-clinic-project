* As an Admin, I can add a new doctor to the system so that the clinic can expand its staff.
* As an Admin, I can view system-wide reports to monitor clinic performance.
* As a Doctor, I can view a list of all my patient appointments so that I can prepare for consultations.
* As a Doctor, I can view patient records so that patients have accurate medication.
* As a Patient, I can search for a doctor by name or specialty so that I can find the right care.
* As a Patient, I can view my upcoming appointments to prepare for my visit.
